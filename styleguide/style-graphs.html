Charts

We developed or adapted a number of useful charts to visualise data and insights from SeMI. They are provided in a small charting library. The examples below are simple use cases of the charts. The demo data used in the examples below can be found in semi-demo-data.js

Sankey Diagram

The Sankey Diagram is useful for showing the connectivity in a graph, i.e. to show through which paths nodes are connected.

Options

width
Width of the chart.

Type: number
Default: 600

height
Height of the chart.

Type: number
Default: 600

HTML / JS code snippet

<div id="sankey-diagram"></div> <script type="text/javascript" src="https://www.gstatic.com/charts/loader.js"></script> <script type="text/javascript" src="/js/semi-demo-data.js"></script> <script type="text/javascript" src="/js/charts.js"></script> <script type="text/javascript"> let data = demoData.sankeyData; let options = { width: 700, height: 300 }; sankeyDiagram('sankey-diagram', data, options); </script>

Force-Directed Graph

The Force-Directed Graph is based on d3-force of d3.js. Tooltips display the keys of the node objects.

HTML / JS code snippet

<div id="force-graph"></div> <script src="https://d3js.org/d3.v4.min.js"></script> <script type="text/javascript" src="/js/semi-demo-data.js"></script> <script type="text/javascript" src="/js/charts.js"></script> <script type="text/javascript"> let data = demoData.forceDirectedGraphData; let options = { width: 600, height: 400, nodeRadius: 5, colorBy: 'class' }; forceDirectedGraph('force-graph', data, options); </script>

Beeswarm Chart

The Beeswarm chart is a mix between the Bubble Chart and the Graph visualisations (Force-Directed Graph and Constraint-Based Layout). As opposed to the Bubble Chart, which is mainly used for aggregated data, the Beeswarm can be used to display the actual data. It arranges single nodes based on selected properties, allowing to get insights about the distribution of the data. A piece of computer code

HTML / JS code snippet

<div id="beeswarm"></div> <script src="https://d3js.org/d3.v4.min.js"></script> <script src="https://cdnjs.cloudflare.com/ajax/libs/p5.js/0.7.2/p5.min.js"></script> <script src="https://cdnjs.cloudflare.com/ajax/libs/p5.js/0.7.2/addons/p5.dom.min.js"></script> <script type="text/javascript" src="/js/semi-demo-data.js"></script> <script type="text/javascript" src="/js/charts.js"></script> <script type="text/javascript"> let data = demoData.beeswarmDataSmall; //let data = demoData.beeswarmDataLarge; var options = { width: 700, height: 700, top: 0, left: 150, bottom: 120, right: 20, bubbleRadius: 4 }; beeswarmChart('beeswarm', data, options); </script>
